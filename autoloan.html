Car Loan App

The Task

I was asked by a new local credit union to add an online car loan application feature to their mobile app. The credit union liked the application seen below from Transcomm and wished to use it as a reference.


Behind the scenes of my design

User research was already done prior to this feature implementation. I kept the different content sections (Loan, Personal, Income, and Finance) but reduced some of the text. This lessens the visual (and cognitive) load for the user (less is more!).
Following that thought, I added drop downs menus and "add more" buttons on several of the questions (aka Progressive Disclosure). This lessens the burden on the user by giving them options to choose from. A progress bar was implemented at the top to encourage the user to continue and lets them know how much is left.
