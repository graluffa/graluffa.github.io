Zoo 404 Page

What was the prompt?

The Zoo has asked me to create an error page (a.k.a. a 404 page) that is user-friendly but also in line with their branding.

What I learned

I learned how to use clever yet clear language (UX writing) to explain why the user received an error and combine it with a design that both keeps the user interested in the site. This involved providing ways to navigate back (search bars and "Home" buttons) as well as an image to accompany. We can maintain our users if we make an unpleasant experience like an error a more pleasant one.
